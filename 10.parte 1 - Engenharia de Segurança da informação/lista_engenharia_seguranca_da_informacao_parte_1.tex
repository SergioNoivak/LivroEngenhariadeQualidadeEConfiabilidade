\begin{block}{Conceitos de Referência}
  \small
  \begin{itemize}
    \item \textbf{CIA Triad}: Confidencialidade, Integridade e Disponibilidade.
    \item \textbf{Malware}: Softwares criados para danos, roubo de dados ou acesso não autorizado.
    \item \textbf{Taxonomia}: Vírus e Trojans (Dependentes) vs. Worms e Bots (Independentes).
    \item \textbf{Ataques}: Passivos (intercepção) vs. Ativos (alteração, fabricação, interrupção).
  \end{itemize}
\end{block}

% --- QUESTÃO 1 ---
\noindent \textbf{1.} Analise o fragmento de pseudocódigo abaixo sobre um mecanismo de propagação:

\begin{lstlisting}
FUNCAO propagar()
    alvos <- ESCANEAR_REDE_LOCAL(porta=445)
    PARA CADA ip EM alvos:
        SE VULNERAVEL(ip):
            COPIAR_PARA_REMOTO(ESTE_ARQUIVO, ip + "/admin/update.exe")
            EXECUTAR_REMOTO(ip + "/admin/update.exe")
        FIM SE
    FIM PARA
FIM FUNCAO
\end{lstlisting}

Com base no comportamento de replicação automática, este malware deve ser classificado como:
\begin{enumerate}[label=(\Alph*)]
    \item Vírus
    \item Worm
    \item Trojan
    \item Ransomware
    \item Spyware
\end{enumerate}

\vspace{0.5cm}

% --- QUESTÃO 2 ---
\noindent \textbf{2.} Analise o fragmento abaixo que descreve uma carga útil (\textit{payload}) maliciosa:

\begin{lstlisting}
PROCEDIMENTO sequestrar_dados(chave_publica)
    PARA CADA arquivo EM "/home/documentos":
        SE EXTENSAO(arquivo) == ".docx" OU ".pdf":
            conteudo <- LER(arquivo)
            criptografado <- ENCRIPTAR_RSA(conteudo, chave_publica)
            SOBRESCREVER(arquivo, criptografado)
        FIM SE
    FIM PARA
    EXIBIR_JANELA("Seus documentos foram bloqueados. Pague para a chave.")
FIM PROCEDIMENTO
\end{lstlisting}

Este comportamento de sequestro de dados via criptografia define um:
\begin{enumerate}[label=(\Alph*)]
    \item Vírus
    \item Worm
    \item Trojan
    \item Ransomware
    \item Spyware
\end{enumerate}
